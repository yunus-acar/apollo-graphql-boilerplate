
# Apollo TypeGraphql Boilerplate




## Installation

Install my-project with npm

```bash
  npm install
  npm run push
  npm run dev
```
    
## Tech Stack


**Server:** Node, Express, Redis, PrismaORM, TypeScript, Apollo, Graphql, TypeGraphql
